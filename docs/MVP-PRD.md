## 1. 项目概述

### 1.1 产品定位
一款**纯浏览器端**运行的免费图片压缩工具，用户无需注册、无需上传图片到服务器，所有压缩处理在本地完成，最大化保护用户隐私。

### 1.2 目标用户
- 运营人员：压缩活动素材、商品图片
- 开发者：压缩网站图片资源
- 普通用户：压缩手机拍照后的大图，方便分享

### 1.3 核心价值
- **免费**：无任何收费功能
- **隐私**：图片不离开浏览器，零上传
- **快速**：纯本地处理，无网络延迟
- **易用**：拖拽上传，即时预览

## 2. 功能需求

### 2.1 P0 — MVP 必做

#### FR-01 图片上传
- 支持拖拽上传（Drag & Drop）
- 支持点击选择文件上传
- 支持键盘粘贴上传（Ctrl+V）
- 支持批量上传（多张图片同时上传）
- 支持格式：JPEG、PNG、WebP、GIF、BMP
- 单张图片大小限制：50MB

#### FR-02 压缩参数设置
- **输出格式**：WebP（默认）、JPEG、PNG
- **压缩质量**：滑块控制，范围 1-100，默认值 80
- **尺寸调整**（可选）：
  - 自定义宽高像素值
  - 锁定/解锁宽高比
  - 不填则保持原始尺寸

#### FR-03 压缩处理
- 使用 Canvas API 进行本地压缩
- 质量参数实时生效，调整后自动重新压缩
- 批量处理：所有图片统一使用当前参数
- 处理中状态提示（loading 动画）

#### FR-04 压缩结果展示
- **前后对比**：滑动对比组件（Before / After）
- **统计信息**：
  - 原始大小
  - 压缩后大小
  - 节省比例（百分比）
  - 总计汇总（全部图片的汇总数据）
- 节省比例颜色标识：
  - 绿色：> 50%
  - 黄色：20% ~ 50%
  - 红色：< 20%

#### FR-05 下载
- **单张下载**：点击下载按钮，保存压缩后的图片
- **批量下载**：一键打包所有压缩结果为 ZIP 文件下载
- 下载文件命名规则：`原文件名.目标格式扩展名`

#### FR-06 图片管理
- 单张删除（带移除动画）
- 一键清空全部图片

## 3. 非功能需求

### 3.1 性能
- 单张 10MB 图片压缩时间 < 3 秒（中端设备）
- 10 张图片批量压缩可正常处理，不卡顿
- 滑块调整后防抖 300ms 再重新压缩

### 3.2 兼容性
- 浏览器支持：Chrome 80+、Firefox 78+、Safari 14+、Edge 80+
- 移动端适配：iOS Safari、Android Chrome
- 响应式设计：桌面端（>768px）、移动端（<768px）

### 3.3 安全与隐私
- **零服务端存储**：所有处理在浏览器 Canvas 中完成
- **零网络传输**：图片数据不会发送到任何服务器
- **隐私声明**：页面明确展示"图片不会被上传"提示

### 3.4 部署
- 平台：Cloudflare Pages
- 方案：纯静态页面（单 HTML 文件）
- 构建步骤：无需（直接部署 `index.html`）
- CDN 加速：Cloudflare 全球节点自动加速
- HTTPS：Cloudflare 自动提供免费证书
- 自定义域名：支持绑定

## 4. 技术方案

### 4.1 技术栈

| 层 | 技术选型 | 说明 |
|---|---|---|
| 页面结构 | 原生 HTML5 | 无框架依赖，单文件方案 |
| 样式 | 原生 CSS3 | 暗色主题，CSS 变量管理 |
| 交互逻辑 | 原生 JavaScript (ES6+) | 无构建工具 |
| 图片压缩 | Canvas API + toBlob | 浏览器原生能力 |
| ZIP 打包 | JSZip (CDN) | 批量下载打包 |
| 文件下载 | FileSaver.js (CDN) | 跨浏览器下载兼容 |

### 4.2 文件结构
```
image-compressor/
└── index.html          # 完整单页应用（含内联 CSS + JS）
```

### 4.3 核心流程

```
用户上传图片 → 读取为 ObjectURL → 绘制到 Canvas → 
→ 应用质量/格式/尺寸参数 → toBlob 导出 → 
→ 展示对比 + 统计 → 用户下载
```

## 8. 风险与应对

| 风险 | 影响 | 应对 |
|------|------|------|
| 大图片（>30MB）压缩慢 | 用户体验差 | 添加进度提示 + 建议用户先调小尺寸 |
| Safari Canvas 兼容性问题 | 部分功能异常 | 测试覆盖，必要时 polyfill |
| PNG 有损压缩效果不明显 | 用户困惑 | UI 提示 PNG 为无损格式，建议选 WebP/JPEG |
| JSZip CDN 加载失败 | 批量下载不可用 | 提供备选 CDN 或内联 |
