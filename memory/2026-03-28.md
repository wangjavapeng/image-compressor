### Image Compressor 项目分析

服务器上的图片压缩项目 `/root/Projects/image-compressor`

**技术栈**: Next.js 15 + React 18 + Tailwind 4 + Cloudflare Workers + D1

**核心特性**:
- 浏览器本地压缩（Canvas API），不上传服务器
- 支持 WebP/JPEG/PNG，可调尺寸和质量
- 积分系统：匿名 3 次，登录 30 积分，VIP 无限
- Google OAuth + JWT Session
- PayPal 支付 ($1 无限套餐)

**关键文件**:
- `src/lib/compressor.ts` - 压缩算法
- `src/hooks/useImageCompressor.ts` - 核心状态管理
- `src/app/api/points/` - 积分扣费/匿名使用
- `migrations/` - D1 数据库 schema
